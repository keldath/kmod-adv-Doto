basic limited religions v1.3b by modifieda4 12/29/11

requires BUG 4.4 installed as standalone mod

installation instructions:

put "Basic Limited Religions.xml" in "config" folder of BUG

extract python files to "python/contrib" folder of BUG (sub folders are allowed)

alter BUG "init.xml" by adding the line in the appropriate section:

<load mod="Basic Limited Religions"/>
